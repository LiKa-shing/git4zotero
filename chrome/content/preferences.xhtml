git4zotero 使用本机 Git 在 Zotero 配置目录中保存论文附件版本。 Git 可执行文件路径 测试 Git 留空表示使用系统 PATH 中的 git。Windows 可填写 C:\Program Files\Git\cmd\git.exe。 当前输入：git 已保存路径：尚未保存 尚未测试 Git。 默认版本说明 创建版本时说明为空，将使用该默认文本。 恢复旧版本前自动创建安全版本 建议保持开启，恢复前会先保存当前文件，降低误覆盖风险。 插件版本 0.2.0 支持格式 .docx 正文级识别；.doc 文件级识别 数据目录 Zotero 配置目录中的 git4zotero 文件夹 是否跟踪某个条目由条目列表右键菜单中的“论文版本”操作控制；右侧面板仅显示状态。
